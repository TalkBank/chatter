# E517: E517: @ID age field does not match a legal CHAT date pattern

> ✅ Active; This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

The @ID header's fourth field (age) must conform to one of the three legal CHAT date patterns defined by CLAN's authoritative depfile.cut:
