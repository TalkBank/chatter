# E549: E549: Duplicate speaker declaration

> ✅ Active; This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

The same speaker code is declared more than once in the @Participants header. Each participant must be declared exactly once; a repeated speaker code is a declaration error.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child, CHI Target_Child
@ID:	eng|corpus|CHI|||||Target_Child|||
*CHI:	hello .
@End
```
