# E514: E514: Empty corpus field in @ID

> ✅ Active; This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

The corpus field (2nd field) of an @ID header is blank. The @ID header is lang|corpus|code|age|sex|group|SES|role|education|custom|, and the corpus name is required: a blank corpus is invalid.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child
@ID:	eng||CHI|||||Target_Child|||
*CHI:	hello .
@End
```
