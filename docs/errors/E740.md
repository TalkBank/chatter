# E740: E740: Phoaln model side does not reproduce the mod word

> ✅ Active; This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

Concatenating the model (left) sides of , skipping ∅, must reproduce the word.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child
@ID:	eng|corpus|CHI|||||Target_Child|||
*CHI:	cat .
%mod:	kæt
%pho:	kæ
%xphoaln:	k↔k,æ↔æ
@Comment:	ERROR: the %xphoaln model side concatenates to 'kæ', which does not match %mod 'kæt'
@End
```
