# E741: E741: Phoaln actual side does not reproduce the pho word

> ✅ Active; This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

Concatenating the actual (right) sides of , skipping ∅, must reproduce the word.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child

%mod:	kæ
%pho:	kæt
%xphoaln:	k↔k,æ↔æ
@Comment:	ERROR: the %xphoaln actual side concatenates to 'kæ', which does not match %pho 'kæt'
@End
```
