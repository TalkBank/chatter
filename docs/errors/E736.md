# E736: E736: Illegal syllable constituent code

> ✅ Active; This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

Constituent codes on / must be one of O N C L R E A D U.

## Examples

```chat

@Begin
@Languages:	eng

%pho:	kæt
%xphosyl:	k:Oæ:Nt:Z
@Comment:	ERROR: 'Z' is not a legal constituent code (legal: O N C L R E A D U)
```
