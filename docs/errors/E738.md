# E738: E738: Phosyl does not reproduce the pho word

> ✅ Active; This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

Stripping :CODE from each %xphosyl unit must reproduce the corresponding %pho word. A pause filler ((.), (..), (...)) on %xphosyl must mirror the same pause token as the %pho word at that position.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child
@ID:	eng|corpus|CHI|||||Target_Child|||
*CHI:	cat .
%pho:	kæt
%xphosyl:	k:Oæ:N
@Comment:	ERROR: stripping codes from %xphosyl gives 'kæ', which does not match %pho 'kæt'
@End
```
