# E739: E739: Phoaln pair is malformed

> ✅ Active; This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

Every %xphoaln pair has exactly one ↔ with a non-null phone on at least one side.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child
@ID:	eng|corpus|CHI|||||Target_Child|||
*CHI:	cat .
%mod:	kæt
%pho:	kæt
%xphoaln:	∅↔∅,k↔k,æ↔æ,t↔t
@Comment:	ERROR: the '∅↔∅' pair is never legal (both sides null)
@End
```
