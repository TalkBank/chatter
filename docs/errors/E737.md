# E737: E737: Modsyl does not reproduce the mod word

> ✅ Active; This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

Stripping :CODE from each %xmodsyl unit must reproduce the corresponding %mod word. A pause filler ((.), (..), (...)) on %xmodsyl must mirror the same pause token as the %mod word at that position.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child
@ID:	eng|corpus|CHI|||||Target_Child|||
*CHI:	cat .
%mod:	kæt
%xmodsyl:	k:Oæ:N
@Comment:	ERROR: stripping codes from %xmodsyl gives 'kæ', which does not match %mod 'kæt'
@End
```
