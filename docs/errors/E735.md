# E735: E735: Syllabification unit is not a phone:CODE pair

> ✅ Active; This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

Every / unit must be one phone, an ASCII ':', then one constituent code.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child
@ID:	eng|corpus|CHI|||||Target_Child|||
*CHI:	cat .
%mod:	kæt
%xmodsyl:	:Oæ:Nt:C
@Comment:	ERROR: the first %xmodsyl unit ':O' has an empty phone before the code
@End
```
