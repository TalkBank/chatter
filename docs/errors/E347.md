# E347: speaker overlap (indexed markers)

> ✅ Active; This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

An indexed top overlap region (e.g., ⌈2...⌉2) on one speaker has no matching indexed bottom overlap region (⌊2...⌋2) from a different speaker, or vice versa. Reported as a warning because some onset-only marking conventions exist.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Child, MOT Mother
@ID:	eng|corpus|CHI|||||Child|||
@ID:	eng|corpus|MOT|||||Mother|||
*CHI:	⌈2hello⌉2 .
*MOT:	hi .
@End
```
